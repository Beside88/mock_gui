What is facebook integration?

What kinds of tokens are there?

There are a different types of tokens. However we will be mainly using user access tokens and page access tokens in this context.

How do you acquire a desired token?

In order to acquire a token the user will need account name/id and the app id.

Additionally user may be asked to accept the persmission request dialog to gain access to the desired pages.
